Website Info

Full Story

I created this website for my network administration class using a bunch of amazing technologies. Using a Digital Ocean droplet running NGINX I am able to serve html webpages to you, the user of my website! The backend of the website uses a Python script which implements the Flask library to route user web requests. This is all being serviced by Gunicorn, which keeps the flask app running at all times so that the website stays up and running.

In order to increase the security of marconi.bio I made sure to configure SSL certificates. To do this, I made use of CertBot, an automated tool for obtaining and managing SSL/TLS certificates. Furthermore, I configured the NGINX configuration file to include a number of security headers. I also had to configure NGINX to serve static elements like images, media, and css files.

As far as front end development of the website goes I used a combination of html and css to style the website to my liking. This is really my first time working with css and I am really glad with how it turned out. If you're an avid user of the desktop application Discord you'll notice I took inspiration from their color scheme.

The purpose of this website is to chronicle my journey into cybersecurity. I plan on writing blog posts from week to week to show others my passion. I'm excited to continue work on this in the coming weeks and months!

Thanks for reading and I hope you have a great day! - Thomas Marconi
